# 月光宝盒 🧿

> 梦幻西游藏宝阁角色智能估价与监控平台

## 功能简介

- **角色估价**：基于市场数据对藏宝阁在售角色进行多维度智能估价，评估修炼、门派、成就、法宝、装备等14个维度
- **价格监控**：设置监控任务，自动发现低价笋货并推送通知
- **数据采集**：定时采集藏宝阁在售/已售角色数据，构建本地价格数据库
- **单角色估价**：通过 Chrome 扩展一键获取角色数据并实时估价

## 快速开始

### 环境要求

- Python 3.11+
- Windows（DrissionPage 依赖 Edge 浏览器获取登录 Cookie）
- Microsoft Edge 浏览器（需登录藏宝阁账号）

### 安装步骤

**1. 安装依赖**

```bash
# 双击运行，或在命令行执行：
install.bat
```

**2. 启动后端**

```bash
start.bat
```

启动后访问：
- 前端界面：http://127.0.0.1:8001
- API 文档：http://127.0.0.1:8001/docs

**3. 安装 Chrome 扩展（可选，用于单角色估价）**

1. 打开 Chrome/Edge，进入 `扩展管理` → 开启「开发者模式」
2. 点击「加载已解压的扩展程序」，选择 `extension/` 文件夹
3. 在藏宝阁角色详情页点击扩展图标即可一键估价

### Cookie 配置

本项目使用 Edge 浏览器 Debug Port 获取藏宝阁登录 Cookie：

1. 用 Edge 浏览器登录 [藏宝阁](https://xyq.cbg.163.com/)
2. 在后端管理页面点击「刷新 Cookie」，系统会自动从 Edge 获取

> 注意：首次使用需在前端「账号管理」页面扫码登录藏宝阁

## 估价引擎说明

估价引擎（`evaluator/engine.py`）基于以下维度综合评估：

1. **角色基础**：等级、门派、转职（飞升/化圣）
2. **成就系统**：成就点数（109级3元/点，129级6元/点，175级5元/点）
3. **人物修炼**：攻/防/法/抗修炼等级（贵价门派1.0x，辅助门派0.65x，冷门门派0.25x）
4. **宠物修炼**：宠攻/防/法/抗修炼等级
5. **技能系统**：特殊技能、门派技能
6. **装备评估**：装备等级、孔数、宝石
7. **外观系统**：时装、翅膀、骑宠、祥瑞
8. **其他属性**：机缘、乾元丹、仙玉、善恶值
