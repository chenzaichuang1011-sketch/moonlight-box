# 更新日志

本文档记录月光宝盒项目的所有重要变更。

## [v0.1.0] - 2026-04-17

### 新增
- 🎯 **核心功能**：梦幻西游藏宝阁角色智能估价系统
- 📊 **估价引擎 v7**：基于14个维度的多维度智能估价
  - 人物修炼评估（贵价/辅助/冷门门派差异化）
  - 宠物修炼评估
  - 成就系统评估（109级3元/点，129级6元/点，175级5元/点）
  - 技能系统评估
  - 装备评估（等级、孔数、宝石）
  - 外观系统评估（时装、翅膀、骑宠、祥瑞）
- 🔍 **价格监控**：自动发现低价笋货并推送通知
- 📈 **数据采集**：定时采集藏宝阁在售/已售角色数据
- 🌐 **Chrome 扩展**：一键获取角色数据并实时估价
- 🖥️ **Web 界面**：Vue 3 + Element Plus 前端界面
- ⚡ **REST API**：FastAPI 后端，支持 OpenAPI 文档

### 技术架构
- **后端**：FastAPI + SQLAlchemy + SQLite + APScheduler
- **前端**：Vue 3 CDN + Element Plus + ECharts
- **爬虫**：DrissionPage + httpx
- **浏览器扩展**：Chrome Manifest V3
- **认证**：JWT + 密码加密

### 数据源
- 43条真实市场成交记录校准
- 实时采集藏宝阁 API 数据
- 本地价格数据库构建

### 使用限制
- 仅支持 Windows 系统（DrissionPage 依赖 Edge 浏览器）
- 需要 Microsoft Edge 浏览器并登录藏宝阁账号
- 仅供学习研究使用，请遵守藏宝阁使用条款

## 版本说明

### 版本命名规则
遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：
- `主版本号.次版本号.修订号`
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新频率
- 功能性更新：每月发布一次次版本
- 问题修复：根据需要随时发布修订版本
- 重大更新：每季度评估主版本更新
